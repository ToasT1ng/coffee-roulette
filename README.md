# 누가 쏠래 🎲

> 커피 한 잔, 누가 쏠지 정해주는 웹 게임 모음  
> **https://www.yourturn.site/**

친구들끼리, 팀원들끼리 "이번엔 누가 쏴?"를 결정할 때 쓰는 작은 웹앱입니다.  
미니게임을 플레이하면 꼴찌(혹은 당첨자)가 자동으로 뽑힙니다.

---

## 게임 목록

| 게임 | 설명 |
|------|------|
| **터치 추첨** | 여러 명이 동시에 화면에 손가락을 올리면 무작위로 한 명을 선택 |
| **사다리 게임** | 고전 사다리 타기로 꼴찌 결정 |
| **꼴찌 정하기** | 캐릭터 경주로 마지막 도착자 선정 |
| **핀볼** | 핀볼 물리 엔진으로 마지막 생존자 선정 |

---

## 개발 환경

```bash
npm install
npm run dev
```

빌드:

```bash
npm run build
```

---

## 기술 스택

- **React 18** + **Vite**
- **Canvas 2D** — 물리 엔진 직접 구현 (핀볼, 경주 등)
- **Vercel** — 배포 및 Analytics
