# 누가 쏠래 🎲

커피 한 잔, 누가 쏠지 정해주는 웹 게임 모음입니다.

## 게임 목록

- **터치 추첨** — 화면에 손가락을 올리면 무작위로 한 명을 선택
- **사다리 게임** — 사다리 타기로 꼴찌 결정
- **꼴찌 정하기** — 캐릭터 경주로 마지막 도착자 선정
- **핀볼** — 핀볼 물리 엔진으로 마지막 생존자 선정

## 개발 환경

```bash
npm install
npm run dev
```

## 기술 스택

- React + Vite
- Canvas 2D (물리 엔진 직접 구현)
